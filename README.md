# Заказ видео
## Очереди
**Стоимость:**<br>
- 0 очередь — 10 руб/мин<br>
- 1 очередь — 20 руб/мин<br>
- 2 очередь — 40 руб/мин<br>
- 3 очередь — 80 руб/мин<br>

Формула: **10 * 2^n руб/мин**<br>

где n — номер очереди.<br><br>

1 USD == 100 руб<br>
1 EUR == 100 руб<br>

*Оставляю за собой право по умолчанию смотреть в ускорении x1.25.*<br>
*Пожалуйста, пишите с донатом номер очереди, чтобы ускорить внесение в очередь*

## Скип 
Cумма доната за видео + 10%

*Если за видео заплатили 100 рублей, то скип будет 110, вне зависимости от того, насколько видео было просмотрено.*

## Инстант
Если заказать видео по очереди выше актуальной на 2, заказанное видео смотрится сразу.<br><br>
К примеру, если есть очередь из видео по 0 очереди и сейчас смотрится видео по 0 очереди, то:<br>
- Заказ по 1 очереди будет просмотрен после просмотра просматриваемого сейчас видео по 0 очереди.<br>
- Заказ по 2 очереди поставит на паузу просматриваемое сейчас видео по 0 очереди и будет начат просмотр видео по 2 очереди.

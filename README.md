# Заказ видео
## Очереди
**Стоимость считается по следующей формуле:<br>

**10 * 2^n руб/мин**<br>

где n — номер очереди

**Отсюда получатся, что:**<br>
0 очередь — 10 руб/мин<br>
1 очередь — 20 руб/мин<br>
2 очередь — 40 руб/мин<br>
3 очередь — 80 руб/мин<br>
И так далее...

*Оставляю за собой право по умолчанию смотреть в ускорении x1.25.*

## Скип 
Cумма доната за видео + 10%

*Если за видео заплатили 100 рублей, то скип будет 110, вне зависимости от того, насколько видео было просмотрено.*
